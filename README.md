## Dependencies

* [Flutter 2.5](https://flutter.dev/)

## Getting started

#### 1. [Setup Flutter](https://flutter.dev/docs/get-started/install)

#### 2. Clone the repo

```sh
$ git clone https://github.com/iwilfried/Flutter-Quiz-App.git
```

#### 3. Run the app

```sh
$ cd Flutter-Quiz-App/flash_card
$ flutter run
```
